# Automatic-stamping-King-of-Time-
Macユーザーが起動した時に自動でKing of Timeに出勤報告出来るように作りました。
右にあるreleasesからexeとenvファイルをダウンロードしてください。
envファイルにはdivision_codeとEmployeeIDとtokenを入力する必要があります。
